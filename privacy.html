Privacy Information

Last Updated September 12, 2025

Below is a list of the types of data we collect and how you can request removal.

  • If you only visit the website (without taking actions such as logging in or submitting an application), the only information we collect automatically is your browser user agent string. This data is used for basic bot and abuse detection and is purged regularly (approximately every 3 months). We do not store IP addresses in our primary Supabase database for simple site visits.
  • If you submit an application, the questions you answer and the application content are saved so we can review and process your submission. For spam and abuse protection we may temporarily process an IP address in server memory or within short-lived logs, but we do not persist raw IP addresses or IP hashes in the Supabase database. Any ephemeral IP data used for automated checks is deleted shortly after the check (typically within a few minutes).
  • If your application is accepted and you create an account, your account username and password are stored in Supabase (passwords are hashed using Argon2). Account creation metadata (creation time, avatar, and related on-site data generated at creation) is kept. We do not attach or store your IP address to your account record in Supabase. Any short-term connection metadata used for rate-limiting or automated abuse detection is not linked to accounts and is removed promptly.
  • If you log in to an account, we authenticate your session and may use short-lived, ephemeral logs to detect multiple failed attempts or suspicious behaviour. These logs are not stored in the primary user database and are not linked to your account permanently. We do not persist IP addresses in Supabase as part of login records.
  • If you join a game, your session uses a JSON Web Token (JWT) to authenticate with the game server. The JWT contains session identifiers and authentication claims only — we do not embed your IP address inside the JWT. Keep your session tokens private; sharing a valid token can allow others to use your session.

Any text you directly enter on the site (descriptions, status updates, comments) is stored. We keep each status update permanently, and only your most recent profile description is shown publicly. Outfits you create, your latest avatar and thumbnails, and any uploaded media (images, etc.) are stored in our object storage. Uploaded content may be removed if it violates our Terms of Service or if you request deletion.

OLDRBX uses third-party services such as hCaptcha, Netlify, and Supabase. These services have their own privacy practices — please review their privacy pages for details about how they handle logs and requests.
